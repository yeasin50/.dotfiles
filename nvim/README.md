# nvim config

Neovim setup with plugin + LSP config.

## install

`bash nvim/.config/install.sh`

or:

```
chmod +x nvim/.config/install.sh
./nvim/.config/install.sh
```

## what it installs

- neovim
- fzf
- ripgrep
- npm
- prettier (global npm tool)
- Nerd Font (for icons)

## requirements

- git
- sudo access
- pacman or apt
- internet

## notes

- safe to rerun (skips installed tools)
- npm “funding” message is normal, not an error
